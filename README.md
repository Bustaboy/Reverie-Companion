# Reverie

**The offline NSFW AI companion that feels truly alive.**

> "Your characters will remember everything important from hundreds or thousands of messages ago, grow with you over weeks and months, and feel like real, evolving people — all while running smoothly on an RTX 4070 8GB mobile GPU."

**Reverie** is a fully local, uncensored desktop AI companion focused on deep long-term memory and genuine character growth.

## What is Reverie?

A modern, high-quality companion that prioritizes:

- **True long-term memory** that actually recalls events from months ago
- **Genuine self-learning & growth** (reflection, journaling, personal LoRA training)
- **Modern, warm, beginner-friendly UI**
- Future integration with video generation tools
- Strict optimization for RTX 4070 8GB mobile
